# Workflow: adhda suite - Dashboard de Apoio ao TDAH

## Objetivo
Fornecer um dashboard interativo e recursos de apoio para indivíduos com TDAH, auxiliando na organização, foco e gestão de tarefas. Inclui scripts Python para geração de dados e schemas para estruturação.

## ICP (Ideal Customer Profile)
Indivíduos com TDAH, profissionais de saúde mental, educadores.

## Fases do Workflow

### Fase 1: Coleta de Dados Pessoais (What)
*   **O Quê:** Coleta de informações sobre hábitos, desafios e preferências do usuário.
*   **Como:** Interação com o usuário através de formulários ou questionários guiados.
*   **Output:** Perfil de usuário e dados iniciais para personalização do dashboard.

### Fase 2: Geração e Estruturação do Dashboard (How)
*   **O Quê:** Criação de um dashboard personalizado com base nos dados coletados.
*   **Como:** Utiliza scripts Python (`adhd_dashboard_generator.py`) para processar dados e schemas (`dashboard_schema.json`) para garantir a estrutura correta do dashboard.
*   **Output:** Dashboard interativo (HTML/Webarchive) com visualizações e ferramentas de organização.

### Fase 3: Acompanhamento e Análise (Where)
*   **O Quê:** Monitoramento do progresso do usuário e análise de padrões de comportamento.
*   **Como:** O dashboard oferece funcionalidades de registro de tarefas, acompanhamento de foco e relatórios de produtividade.
*   **Output:** Relatórios de progresso e insights sobre a eficácia das estratégias de apoio.

### Fase 4: Refinamento e Suporte (Why)
*   **O Quê:** Ajustes no dashboard e fornecimento de recursos adicionais com base no feedback e progresso do usuário.
*   **Como:** Utiliza referências (`adhd-design-principles.md`, `sources-and-safety.md`, `workflow-protocol.md`) para garantir a eficácia e segurança das intervenções.
*   **Output:** Dashboard otimizado e acesso a materiais de apoio personalizados.

## Skills Envolvidas (Exemplos)
*   `adhd-desk-dashboard-skill` (skill principal)

## Pontos de Controle
*   Privacidade e segurança dos dados pessoais.
*   Relevância e utilidade das funcionalidades do dashboard.
*   Eficácia das estratégias de apoio ao TDAH.
*   Facilidade de uso e acessibilidade da interface.

## Interdependências
*   Pode integrar-se com ferramentas de calendário ou gerenciamento de tarefas externas.
*   Pode gerar relatórios para profissionais de saúde acompanharem o progresso do paciente.
