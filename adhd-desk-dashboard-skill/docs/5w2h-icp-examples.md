# 5W2H + ICP + Examples

| Element | Answer |
|---|---|
| What | A skill that creates ADHD/TDAH-friendly desk dashboards and execution routines. |
| Why | To reduce cognitive load, ambiguity, and task switching. |
| Who | Solo researchers, creators, founders, consultants, and operators. |
| Where | Desk, paper binder, Linear, Drive, Agenda, Claude/ChatGPT, GitHub. |
| When | When work is complex, fragmented, or hard to restart after interruption. |
| How | Convert raw workflow into printable A4 pages with clear fields and DoD. |
| How much | Starts as a lightweight template; can evolve into PPTX/PDF automation. |

## ICP

| Field | Description |
|---|---|
| Primary ICP | Solo knowledge worker with high-volume ideas and execution friction. |
| Pain | Too many contexts, notes, tools, and unfinished tasks. |
| Desired outcome | A visible daily operating system. |
| Buying/usage trigger | Needs to stabilize sprint execution or content/research pipeline. |

## Examples

| Example | Applicability | Output |
|---|---|---|
| Research workflow | Capture findings without losing source/action context. | Research card and data-clean routine. |
| Content pipeline | Turn one insight into multi-channel posts. | Content creation page and Linear backlog. |
| Admin control | Prevent email/account/agenda drift. | 90-minute ops checklist. |
