# ADHD Desk Dashboard Skill — README

| Section | Description |
|---|---|
| Skill name | ADHD Desk Dashboard Skill |
| Primary purpose | Transform a messy workflow command into a printable, low-cognitive-load operating dashboard. |
| Main user | Solo knowledge worker, researcher, creator, founder, consultant, or operator who needs visible execution structure. |
| Accessibility frame | ADHD/TDAH and dyslexia-friendly execution support: large text, visual blocks, low noise, explicit next actions, and Definition of Done. |
| Core artifact | A4 landscape printable dashboard, usually 6 slides or fewer. |
| Physical workflow | Printed sheets on desk, board, binder, clipboard, or wall. |
| Digital workflow | Linear issues, Drive folders, Calendar blocks, GitHub references, AI command library. |
| Main benefit | Reduces ambiguity and context switching by making work visible, bounded, and checkable. |
| Non-goal | Medical diagnosis, medication guidance, therapy replacement, or clinical assessment. |

## Real Usage Table

| Use Case | Input | Process | Output | Success Criterion |
|---|---|---|---|---|
| Weekly sprint desk board | Raw weekly goals | Convert into Roadmap Mesa | Printable page | User can identify one objective and next action in under 30 seconds. |
| Daily workflow table | Task categories and time boxes | Map capacity and DoD | Capacity table | Every workflow has input, output, save location, and DoD. |
| Data clean routine | Raw notes, chats, files | Classify into FACT/HYPOTHESIS/DECISION/ACTION | DataClean page | Inbox reduced and reusable knowledge saved. |
| Content routine | Topic, channels, reuse plan | Batch and route content | Content workflow page | Content published, URLs saved, metrics captured. |
| Ops admin routine | Admin checklist | Convert into 90-minute loop | Ops page | Login, email, accounts, agenda, and backlog checked. |
| Research capture | Finding or source | Fill one card | Research card | Finding has source, number, hypothesis, decision, and action. |
